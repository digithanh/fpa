## Qu'est-ce qu'un ordinateur ?
```
Un ordinateur PC est constitué d’une unité centrale(UC) qui doit être connectée à des périphériques comme :
o L’écran : permettant de visualiser les résultats demandé
o Le clavier : envoie les informations et formules les demandes travaux
o Souris : est un pointeur physique actionnant des commandes virtuelles
o Imprimante : confirme les résultats sur papier
```
